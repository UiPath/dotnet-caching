# Security Policy

## Supported versions

The latest minor release of UiPath.Caching receives security updates.

## Reporting a vulnerability

**Do not open a public issue for security vulnerabilities.**

Report privately to <SECURITY_CONTACT — TBD: disclosure email or URL>. Include:
- Affected version(s)
- Reproduction steps or proof-of-concept
- Impact assessment

We aim to acknowledge within 3 business days and to publish a fix within 30 days for high-severity issues.
