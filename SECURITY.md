# Security Policy

## Reporting a vulnerability

Do not open a public GitHub issue for security problems. Report suspected vulnerabilities through UiPath's responsible-disclosure process at <https://www.uipath.com/legal/trust-and-security>. Include the affected package version(s), reproduction steps or a proof-of-concept, and an impact assessment.

## Supported versions

The latest released minor version of the `UiPath.Caching.*` packages receives security updates.
